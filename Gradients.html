Linear Gradient

This is a linear gradient from top to the bottom ..To apply the linear gradient,we apply CSS3 styling property background And value linear-gradient(#startOffColor,#EndOfColor);

In this example to the div we have applied styling as :::

background:linear-gradient(white,blue);

Vertical Linear Gradient

In this Gradient,the first arguement of the function linear-gradient is not left to default(i.e. to bottom)

And thus,the gradient is applied to right as follows:: background:linear-gradient(to right,#ABCDEF,#FEDCBA);

The first arguement can have the following values :

  • to bottom
  • to top
  • to right
  • to left
  • to top right
  • to bottom right
  • to bottom left
  • to top left

Multicolored Gradient

This is to show that there is no limit to colors arguement ,we just need to provide the color in order

Here we have :::

background: linear-gradient(to top right,violet,indigo,blue,green,yellow,orange,red);

Radial Gradient

Here we have a radial gradient. This Gradient comes from center to borders of the div

In this example :::

background: radial-gradient(white,black,white,black,white,black);
